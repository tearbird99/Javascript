### 🔖 개요
자바스크립트에서 배열의 요소를 읽고 쓰는 법, 배열의 요소들을 순회하는 법을 알아본다.

---

### ⭐ 배열 요소 읽기와 쓰기
```javascript
let a = ["world"];
let value = a[0]; // value = "w";
a[1] = 3.14; // a = ["world", 3.14];

let b = [true, false];
a[2]; // undefined; 이 인덱스에는 요소가 없습니다
a[-1]; // undefined; 이런 이름의 프로퍼티는 없습니다 (인덱스는 항상 0 이상의 정수)
```
```javascript
let c = new Array(5); // c.length = 5
a = []; // a.length = 0
a[1000] = 1; // a.length = 1001
```

```javascript
let a = [1, 2, 3, 4, 5];
a.length = 3; // a = [1, 2, 3]
a.length = 0; // a = []
a.length = 5; // a = new Array(5)의 결과와 같아짐
```

### ⭐ 배열 순회
```javascript
let letters = [..."Hello World"];
let str = "";

for (letter of letters) {
  str += letter;
}
console.log(str); // "Hello World"
```
```javascript
let everyother = "";
for (let [index, letter] of letters.entries()) {
  if (index % 2 === 0) everyother += letter;
}
console.log(everyother); // HloWrd
```

#### 🌟 forEach() 메서드
- forEach()는 배열을 순서대로 순회하며 배열 인덱스를 함수의 두 번째 인자로 전달한다.
- for-of 루프와 달리 forEach()는 성긴 배열을 인식하고, 존재하지 않는 요소에 대해서는 함수를 호출하지 않는다.
```javascript
배열명.forEach((요소, 인덱스) => {
  // 실행할 코드
});
```
```javascript
let numbers = [10, 20, 30];
numbers.forEach((num, i) => {
  console.log(`${i}번째 숫자는 ${num}.`);
});
// 0번째 숫자는 10.
// 1번째 숫자는 20.
// 2번째 숫자는 30.

let letters = [..."Hello World"];
let uppercase = "";
letters.forEach((letter) => {
  uppercase += letter.toUpperCase();
});
console.log(uppercase); // HELLO WORLD       
```
