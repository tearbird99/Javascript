### ⭐ 기본 원칙
> - 키워드(예약어)와 같은 이름을 사용할 수 없다.
> - 숫자로 시작할 수 없다.
> - 특수문자는 _와 $만 허용한다.
> - 공백 문자를 포함할 수 없다.
> - 여러 단어를 합할 경우에는 카멜 케이스(camelCase)를 따른다.

---

### ⭐ 추가 예시
```javascript
// 대문자로 시작하면 클래스
Car
People

// 상수 (변경 X)
MAX_VALUE
NAME
```
```javascript
num // 변수
alert('warning...') // 함수
Array.length // 속성
Math.abs(-730) // 메서드
```
