### 🔖 개요
자바스크립트에서 반복문은 동일한 작업을 여러 번 수행해야 할 때 코드를 효율적으로 줄여주는 도구이다. 대표적인 3가지 반복문에는 for, while, do-while 문이 있다.

---

### ⭐ for 문
반복 횟수가 명확할 때 주로 사용한다. 초기화, 조건식, 증감식을 한 줄에 작성하여 가독성이 좋다.
- **구조**: for (초기화; 조건식; 증감식) { ... }
```javascript
for (let i = 0; i < 5; i++) {
  console.log(`${i}번째 반복 중`); // 0부터 4까지 출력
}
```
- **작동 순서**:
**1. let i = 0**: 변수 초기화 (처음 한 번만 실행)
**2. i < 5**: 조건식 검사 (참이면 실행, 거짓이면 종료)
**3. { ... }**: 코드 블록 실행
**4. i++**: 증감식 실행 후 다시 2번으로 이동

---

### ⭐ while 문
조건이 참(true)인 동안 계속해서 반복한다. 반복 횟수가 유동적이거나 특정 조건에 도달할 때까지 실행해야 할 때 적합하다.
```javascript
let count = 0;

while (count < 3) {
  console.log(`현재 카운트: ${count}`);
  count++;
}
```

---

### ⭐ do-while 문
while 문과 유사하지만, **조건을 검사하기 전에 코드를 최소 한 번은 무조건 실행**한다는 점이 다르다. 또한 do-while 문은 반드시 세미콜론(;)으로 끝나야 한다.
```javascript
let i = 10;

do {
  console.log("이 문장은 조건이 거짓이라도 한 번은 실행됩니다.");
  i++;
} while (i < 5); 

// 결과: 한 번 출력 후 종료
// 이 문장은 조건이 거짓이라도 한 번은 실행됩니다.
```

---

### ⭐ 반복문 제어 키워드
- **break**: 반복문을 즉시 종료하고 빠져나온다.
- **continue**: 현재 반복 회차를 건너뛰고 다음 반복(증감식 또는 조건 검사)으로 넘어간다.
```javascript
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // 3은 건너뜀
  if (i === 5) break;    // 5가 되면 종료
  console.log(i); // 1, 2, 4
}
```
