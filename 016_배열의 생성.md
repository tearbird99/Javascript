### 🔖 개요
배열 역시 객체와 같이 여러 방법으로 생성할 수 있다.

---

### ⭐ 배열의 생성

#### 🌟 배열 리터럴
가장 직관적이고 흔히 사용하는 방법이다.
```javascript
const arr = []; // 빈 배열
const fruits = ['apple', 'banana', 'cherry'];
const points = [
  [1, { x: 1, y: 2 }],
  [2, { x: 3, y: 4 }],
];

const sparse = [1, , 3]; // 요소 사이에 갭이 있는 성긴 배열(Sparse Array)
```

---

#### 🌟 분해 연산자 (...) 활용
```javascript
const original = [1, 2, 3];
const copy = [...original, 4, 5]; // [1, 2, 3, 4, 5]

const str = "Hi";
const charArray = [...str]; // ['H', 'i']
```

---

#### 🌟 Array() 생성자 함수
```javascript
const arr = new Array(); // 인자 없이 빈 배열 ([]과 동일)
const arr1 = new Array(3);     // [empty × 3] - 지정된 길이를 가진 희소 배열
const arr2 = new Array(1, 2, 3); // [1, 2, 3]
```

---

#### 🌟 팩토리 메서드 Array.of(), Array.from()
생성자 함수의 단점을 보완하거나 특정 객체를 배열로 변환할 때 사용한다.
- **Array.of()**: 인자의 개수와 상관없이 전달된 인자를 모두 요소로 가지는 배열을 만든다.
```javascript
const arr = Array.of(3); // [3]
const arr1 = Array.of(1, 2, 3); // [1, 2, 3]
```
- **Array.from()**: 유사 배열 객체(Array-like object)나 이터러블 객체를 진짜 배열로 변환한다. (예.문자열->배열)
```javascript
// 유사 배열(length를 가진 객체) 변환
const arr = Array.from({ length: 3 }, (v, i) => i); // [0, 1, 2]

const str = "Hi";
const charArray = [...str]; // ['H', 'i']
const charArray2 = Array.from(str); // ['H', 'i']
```
