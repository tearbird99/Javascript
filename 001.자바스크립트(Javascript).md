### 🔖 개요
```Javascript
console.log("Hello, World");
```
한 마디로 **HTML과 CSS로 구성된 웹 페이지를 동적으로 만들어주는 언어**라고 할 수 있다. HTML로 웹의 구조를 세우고, CSS로 웹을 디자인한다. 그리고 Javascript는 그렇게 만들어진 웹을 동작하게 한다. 비유하자면 HTML은 자동차의 뼈대, CSS는 자동차의 외관, JavaScript는 자동차의 동력원인 엔진이라고 볼 수 있다.

---

### ⭐ 특징
- **인터프리터 언어:** 별도의 컴파일 과정 없이 브라우저가 해석해 실행 (원시 코드를 기계어로 변환하지 않고 한 줄씩 해석하여 바로 명령어를 실행)
- **동기적 언어:** 호출 스택이 하나뿐이라 한 번에 하나의 작업만 순서대로 처리
- **동적 타입 언어:** 변수의 자료형을 명시하지 않아도 자동으로 판단
- **프로토타입 기반 언어:** 설계도(클래스) 없이, 이미 존재하는 객체(프로토타입)를 복제하거나 참조해서 새로운 객체를 제작
- **이벤트 기반 프로그래밍:** 사용자의 클릭, 입력 등 이벤트에 반응하여 실행

---

### 📚 활용
- **웹 프론트엔드**: HTML, CSS와 함께 웹 UI를 구성
- **백엔드**: Node.js를 활용해 서버 개발
- **모바일 앱 개발**: React Native, Ionic 등과 같이 사용해 모바일 앱 개발
- **데스크톱 앱 개발**: Electron으로 크로스 플랫폼 앱 개발
- **게임 개발**: 2D 또는 3D로 브라우저 게임 개발
